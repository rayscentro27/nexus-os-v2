# Alpha Free Search Options Review

No free provider is claimed active in v1. DuckDuckGo Instant Answer JSON was the first choice because it avoids HTML scraping, but project-runtime verification returned an empty body even though command-line curl worked. That inconsistency makes it too brittle to enable.

SearXNG is the implemented gated backend option when Ray supplies an approved instance URL. Firecrawl would add crawling/extraction beyond this sprint and remains disabled. Brave requires an approved server-side key. No dependency was installed.
