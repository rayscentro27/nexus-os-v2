# Alpha Provider Route Trace

Every Alpha response records message/time/mode, selected and actual provider, backend source, model, session memory, reports, web provider/status/query/result count, fallback reason, call estimates, cost label, and hard `supabase_used: false` / `client_data_used: false` fields.

The collapsible source line uses plain language, for example: `Response source: Groq backend · Memory: session memory · Web: off · No Supabase · No client data`.
