# Credit Report Review Flow

- Flow helper: `src/lib/creditReportReviewFlow.ts`
- Credit Profile is report-first: `True`

## Entry Options

1. Upload Credit Report.
2. Connect Credit Monitoring Resource, gated until a secure provider exists.
3. I Need Help Getting My Report.
4. Manually Add Negative Item with non-sensitive fields only.

## After Report / Item Exists

Clyde shows items to challenge, utilization improvement options, missing evidence, dispute letter options, specialist review needs, and funding impact.
