# Dispute Letter Options

Supported reasons:

- Not mine
- Incorrect balance
- Incorrect dates
- Duplicate
- Paid/settled reporting wrong
- Late payment wrong
- Unauthorized inquiry
- Personal information error
- Verify/validate this item
- Outdated
- Mixed file
- Method of verification follow-up
- Other
- I am not sure

## Option Families

- Bureau dispute
- Furnisher direct dispute
- Collector validation
- Method of verification
- Reinvestigation
- Goodwill option as non-dispute strategy
- CFPB escalation prep for repeated unsupported verification
- Personal information correction
- Duplicate account clarification

Each option includes when to use it, why it fits, evidence needed, caution language, draft body, risk level, and next action.
