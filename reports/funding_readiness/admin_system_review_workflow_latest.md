# Admin System Review Workflow

The admin route remains compatible and is titled **Credit & Funding Readiness Review**. Admins can queue analysis, refresh verified parser/system results, review live counts, and see system recommendations with confidence and Tier 1/Tier 2 impact.

Available exception-review decisions are Confirm Recommendation, Edit, Reject, Request Client Evidence, Add Manual Item, and Prepare Draft Letter. Draft preparation stays disabled unless the recommendation is both confirmed and letter-eligible. Decision UI is currently an in-session review control; durable per-item decision persistence is the next admin workflow increment.
