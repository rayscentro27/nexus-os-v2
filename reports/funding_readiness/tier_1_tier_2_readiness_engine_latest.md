# Tier 1 / Tier 2 Readiness Engine

The combined evaluator uses Credit Profile data sufficiency, utilization, inquiries, major funding-impact items, and pending reviews with Business Profile completeness. Tier 2 additionally considers operating history, revenue, banking, documents, obligations/industry review inputs.

Outputs include a bounded 0–100 readiness score, allowed readiness status, blockers, next three actions, missing data, and an explanation that the result is not approval. The client Funding Readiness page now shows `Credit Profile + Business Profile = Tier 1 / Tier 2 Funding Readiness`.
