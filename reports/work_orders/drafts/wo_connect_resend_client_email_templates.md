# Work Order: Connect Resend Client Email Templates

**Status:** Draft
**Priority:** Low
**Source:** Client Portal Premium UI Foundation Sprint
**Created:** 2026-07-07

## Objective
Send client-facing emails via Resend for key portal events.

## Scope
- Welcome email on signup
- Document upload confirmation
- Review request confirmation
- Status update notifications
- Invoice/payment receipts

## Acceptance Criteria
- Emails sent via Resend API
- Templates stored and versioned
- Unsubscribe link included
- Client email preferences respected

## Dependencies
- Resend API key configured
- Client email addresses in Supabase
- Email templates designed and approved
