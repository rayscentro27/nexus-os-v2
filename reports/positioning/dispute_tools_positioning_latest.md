# Dispute Tools Positioning

Dispute functionality remains available as a self-service readiness tool. Visible framing now emphasizes Draft Letter Tool, Documentation Preparation, Client Review Required, GoClear Review Required, and mailing only after approval.

Every generated preview carries this outcome-safe notice: “Draft preview only. This document requires review and approval before use. Nexus does not guarantee a specific credit reporting outcome.”

The system does not auto-send. Existing client approval and `approved_for_docupost` gates remain intact. Internal dispute-related types, reason codes, and generator names remain for compatibility.
