# Hermes Alpha Research Inbox

This directory is for approved local Hermes Alpha research artifacts only.

Rules:

- Do not add client data, personal financial data, or customer records.
- Do not add secrets, credentials, API keys, tokens, or environment files.
- Do not add Supabase exports or database dumps.
- Do not add Oanda account, transaction, position, pricing, or credential data.
- Do not add payment, checkout, card, banking, or subscription data.
- Do not add production credentials or private download dumps.
- Do not scan or copy arbitrary repositories into this directory.
- Allowed artifact types are `.md`, `.txt`, `.json`, and `.csv`.
- Treat every artifact as untrusted, read-only evidence requiring validation.
- Artifact presence does not make a claim true or approved.
- Alpha outputs remain draft-only and require Ray Review before any handoff.

Folder README files are policy documentation, not research artifacts. Empty category folders are valid. Do not add mock material here to make Alpha appear active; evaluation fixtures belong under `hermes_alpha/evaluations/fixtures/`.
