# Nexus YouTube to SEO/Affiliate Plan

`scripts/research/youtube_to_seo_affiliate_plan.py` converts YouTube research report candidates into internal SEO and affiliate content plan candidates.

Outputs include:

- source video
- channel
- target keyword
- article angle
- affiliate offer hint
- GoClear offer tie-in
- CTA
- compliance note
- recommended department
- priority score

The script does not publish content, create public pages, send messages, or create Ray Review Queue items for ordinary research.
