# Nexus Long-Term Operating Roadmap

Core rule: a source is not just content. A source is an asset Nexus watches, scores, learns from, and turns into experiments over time.

## Operating Lanes

Lane 1, Autonomous Internal Research:

- watch approved resources
- backfill approved resources
- detect new items
- review transcripts/pages/reports
- score and re-score
- route to departments
- create internal project/experiment cards
- create reports
- update Hermes memory

Lane 2, Approval-Gated Execution:

- publish
- send
- contact
- trade/live execute
- spend money
- schedule persistent automation
- production change
- sensitive/private data in external tools
