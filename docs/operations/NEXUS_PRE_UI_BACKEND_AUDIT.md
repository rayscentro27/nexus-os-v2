# Nexus Pre-UI Backend Audit

`scripts/audit/run_pre_ui_backend_audit.py` checks the backend foundation before the next UI redesign.

Audit areas:

- watched resources
- YouTube metadata connector
- YouTube metadata check
- metadata backfill
- transcript availability/review
- YouTube reports
- Hermes prep
- SEO/affiliate conversion
- content experiments
- Scheduler Approval Center candidates
- Ray Review Queue

The audit reports what is ready, partial, disabled, needs Ray decision, can run autonomously, and should appear in the next UI image set.
